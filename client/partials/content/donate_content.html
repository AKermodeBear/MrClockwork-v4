Mr. Clockwork is a community effort that consists of not only this website but also multiple game servers to host Dominions games as well as the code for the Mr Clockwork bot that helps runs games through Discord.

The funds raised for Mr. Clockwork help keep those servers running as well as provide future funds to keep the system running for up to six months.
